# AI × Web3 School 学习仓库 — Wang

Personal learning journal and proof-of-work for AI × Web3 School.

## 固定入口

- Handbook：https://aiweb3.school/zh/handbook/
- WCB 课程页面：https://web3career.build/zh/programs/AI-Web3-School
- WCB Learning 页面：https://web3career.build/zh/programs/AI-Web3-School#tab=learning
- WCB Agent API 文档：https://web3career.build/llms.txt

## 仓库用途

这个仓库用于沉淀：

- 每日学习记录与打卡草稿：`daily/`
- 课程任务拆解与 proof-of-work：`tasks/`
- AI × Web3 小实验：`experiments/`
- Handbook 反馈：`handbook-feedback/`
- Hackathon 想法、方案与提交材料：`hackathon/`
- 最终提交材料：`submissions/`
- 复用模板：`templates/`

## 隐私与安全提醒

本仓库默认为 **public**，不要提交任何敏感信息，包括但不限于：

- API Key、token、密码、验证码
- 助记词、私钥、Keystore 文件
- 未公开联系方式、内部会议链接
- 他人个人数据或未经许可的截图
- WCB / 第三方平台 secret

如需使用 WCB Agent API，secret 仅保存在本地环境变量或 Hermes secrets，例如：`WCB_AGENT_SECRET_API_KEY`。
